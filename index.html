MAP-E

The objective of this POC is to develop and set up the Middleware Monitoring solution for targeted servers, App endpoints, Custom DB and Application metrics. The solution will also configure alerts and would be integrated with ticket management system for all environments such as dev UAT, pre-production and production.

Business Case

  • Currently the tasks such as disk utilization, CPU utilization, memory, server/application health checks are being checked manually which is a tedious job. To avoid this manual intervention monitoring solution is required which will provide 24/7 monitoring services of all servers and painful areas.
  • Integrating the alert mechanism will trigger attention of the users if any unexpected system behaviour is happening in the environment
  • Integrating with the ticketing tool would help the users to raise the auto ticket(s) if the resources reached its threshold values
