About project

GRM is HPE VM migration cloud based application to migrate Virtual machine which used via cloud based server by creating user profile for serevral migration projects it help to migrate VM on several different OS based application. Unix to linux.

it help migration of vm from one host to another with no downtime. it also help user to generate reports based on Profile, projects and migration it allow to plan for VM for cloud migration.

& plan will help to determine the migration feasibility, validate pre-requisites and recommend migration approach. Based on the feasibility it will allow you to create migration wave plans. it allow user to select source and target based migration. it also facilitates to use several projects on one user profile.
